#KeenKeeper — Keep Your Friendships Alive

KeenKeeper is a web application designed to help users track their communication history and maintain strong relationships with friends. It features status tracking, interaction logging, and insightful analytics.

#Key Features:
Friend Dashboard: View all friends with color-coded statuses (Overdue, Almost Due, On-Track).

Interactive Timeline: Log and filter interactions (Call, Text, Video) with auto-updates.

Friendship Analytics: Visualize communication habits using pie charts.

Responsive UI: Fully mobile-friendly design built with Tailwind CSS.

🛠 Technologies Used
Frontend: React.js
Styling: Tailwind CSS
Visualization: Recharts
Routing: React Router DOM
Icons: FontAwesome.
